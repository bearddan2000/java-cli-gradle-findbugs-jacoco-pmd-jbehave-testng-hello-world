# java-cli-gradle-findbugs-jacoco-pmd-jbehave-testng-hello-world

## Description
A POC app using testng, findbugs,
jacoco, pmd, jbehave framework.

## Tech stack
- java
- gradle
  - testng  
  - jbehave
  - jacoco
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
